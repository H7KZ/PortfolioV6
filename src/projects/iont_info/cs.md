IONT Info je fullstack webová platforma vyvinutá pro správu a monitoring sítí nabíječek elektromobilů pro IONT Tech. Výzvou bylo postavit celý systém od nuly — infrastrukturu, backend i frontend — a zároveň integrovat hardware nabíječek na bázi Raspberry Pi přes API firmware týmu. Výsledkem je cloud-native platforma na mikroservisové architektuře, která se škáluje napříč sítěmi nabíječek a poskytuje ovládání v reálném čase přes responzivní mobilní rozhraní.
