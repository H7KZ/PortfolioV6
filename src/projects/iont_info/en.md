IONT Info is a fullstack web platform built to manage and monitor electric vehicle charger networks for IONT Tech. The challenge was building the entire system from the ground up — infrastructure, backend, and frontend — while integrating with Raspberry Pi-based charging hardware via a firmware team API. The result is a cloud-native, microservices-based platform that scales across charging networks and provides real-time control through a responsive mobile-first interface.
