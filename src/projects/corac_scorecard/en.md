Scorecard is a web application built for CORAC Engineering that functions as an interactive spreadsheet for cyber auditors — purpose-built for risk analysis data entry and report generation. The challenge was creating a tool that could replace ad-hoc spreadsheets while adding professional output capabilities without requiring a backend database. The result is a portable, offline-capable application that exports audit results directly to white-labeled PDFs and supports NIS2 compliance workflows.
