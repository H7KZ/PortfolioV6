Disenchantment je Java plugin pro Minecraft postavený na Spigot API, který řeší dlouhodobé vanilla omezení: jakmile enchant aplikujete, je permanentní. Plugin zavádí novou mechaniku umožňující hráčům odebírat a získávat enchanty z předmětů přes přepracovanou interakci s kovadlinou, přidávající skutečnou strategickou hloubku ke správě inventáře. Postaven s důrazem na kompatibilitu a výkon, funguje napříč verzemi Minecraft 1.18–26.x bez závislostí a je open source na GitHubu.
